# HA-Ouman-23H
Home Assistant integration for Ouman H23 heating controller using Modbus RTU/TCP
## Hardware
- **Server:** Dell OptiPlex 7050
- **Controller:** Ouman H23
- **Gateway:** Waveshare RS485 to RJ45 Ethernet Converter

## Connection Settings
- **IP Address:** 192.168.x.x
- **Port:** 502
- **Modbus Slave ID:** 1
